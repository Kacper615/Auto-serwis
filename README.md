# Projekt: Auto-Serwis

### Autor: Kacper Kopaczewski
**Przedmiot:** Technologie Internetowe
**Technologie:** HTML5, CSS3

## Opis projektu
Projekt przedstawia profesjonalną stronę internetową warsztatu samochodowego zlokalizowanego w Białymstoku. Witryna została zaprojektowana z myślą o czytelności i nowoczesnym wyglądzie.

## Spełnione wymagania:
* **Struktura HTML:** Poprawna semantyka, tytuły oraz meta opisy.
* **3 podstrony:** Strona główna, Oferta oraz Kontakt.
* **Nawigacja:** Menu w formie listy (ul/li).
* **Stylizacja:** Arkusz stylów w oddzielnym pliku `style.css`.
* **Multimedia:** Wykorzystanie grafik technicznych oraz interaktywnej mapy Google.
* **Responsywność:** Strona dostosowana do urządzeń mobilnych.
